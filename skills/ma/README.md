# China M&A Legal Skills

面向中国并购法律实务的 AI Agent Skills 集合，覆盖股权并购、资产并购、跨境并购的全流程：尽职调查 → 交易方案设计 → 交易文件起草 → 交割与整合。

## Skill 列表

| Skill | 标签 | 说明 |
|:---|:---|:---|
| [并购法律工作总控](skills/ma/并购法律工作总控/) | 总控 | 统一入口，语义路由、事项隔离、材料读取复查、法规校验、来源边界和质量门调度 |
| [并购尽职调查](skills/ma/并购尽职调查/) | DD | 法律尽职调查方案制定、清单生成、发现问题分析、DD报告撰写 |
| [并购交易方案设计](skills/ma/并购交易方案设计/) | 方案 | 交易结构设计、税务筹划框架、支付安排、交割条件设计 |
| [并购交易文件](skills/ma/并购交易文件/) | 文件 | SPA/SHA/APA 起草、核心条款谈判要点、附件体系 |
| [并购交割与整合](skills/ma/并购交割与整合/) | 交割 | 交割条件核查、交割流程管理、交割备忘录、整合风险评估 |
| [跨境并购专项](skills/ma/跨境并购专项/) | 跨境 | ODI备案、外资审查、跨境支付、适用法律冲突 |

## 快速开始

```bash
git clone https://github.com/your-username/ma-legal-skills.git
cp -r skills/ma/ ~/.qclaw/skills/ma/
```

## 本地配置

使用前请配置：
- 律师姓名、律所名称、联系信息
- 工作区与业务文件路径
- 尽职调查核查表库目录
- 交易文件模板库目录
- 法律法规数据库凭证
