# 来源边界协议

## 输出标注规则

所有分析、报告、文件必须标注以下来源分类：

### 已核验
- 直接读取的用户文件，内容可验证
- 通过法规数据库核验的现行法规
- 用户明确确认的事实

### 未核验
- OCR识别结果，可能与原文不一致
- 模型根据已有知识库生成的法规引用，未经数据库核验
- 网页搜索结果，未与官方源交叉验证

### 缺口
- 交易方未提供的材料
- 目标公司未披露的信息
- 需要进一步尽调才能确认的事项

## 输出模板

每次输出末尾应包含：

```markdown
---
### 来源边界

**已核验**：[列出已核验内容]
**未核验**：[列出未核验内容及原因]
**缺口**：[列出缺失材料和待确认事项]
```
