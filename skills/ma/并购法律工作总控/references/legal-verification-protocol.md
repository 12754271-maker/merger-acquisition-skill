# 法规校验协议

## 核验范围

并购交易涉及的核心法律法规包括但不限于：

- 《中华人民共和国公司法》（2023修订）
- 《中华人民共和国证券法》
- 《上市公司收购管理办法》
- 《企业国有资产交易监督管理办法》（32号令）
- 《外商投资法》及其实施条例
- 《外商投资准入特别管理措施（负面清单）》
- 《关于外国投资者并购境内企业的规定》（10号文）
- 《企业境外投资管理办法》（11号令）
- 《反垄断法》及经营者集中申报规定
- 行业特别监管规定（金融、能源、电信等）

## 核验流程

1. **名称编号内容核验**：法规名称、条文编号、条文内容三者一致
2. **现行有效性核验**：确认法规现行有效，未废止或修订
3. **形成法规校验摘要**：
   - 引用法规名称 + 条款编号
   - 核验结果（已核验 / 未核验 / 存疑）
   - 时效性状态（现行有效 / 已修订 / 已废止）
   - 未核验原因（无权限访问数据库 / 网络不可用等）

## 禁止事项

- 不得用模型记忆替代法规检索
- 不得将未核验法规标注为已核验
- 不得遗漏影响交易的重大法规变化
