## 这是什么？

一套装在 QClaw AI 助手里的「并购法律技能包」。你跟它说人话，它自动帮你干活：

| 说这句 | 它干这个 |
|--------|----------|
| 「帮我出个制造业尽调清单」 | → 自动生成法律尽调清单 |
| 「我是买方，起草SPA」 | → 起草股权转让协议核心条款 |
| 「收购新加坡公司流程」 | → 分析 ODI 备案 + 外资审查 |
| 「股权收购和资产收购怎么选」 | → 对比分析 + 税务框架 |

## 六个模块，全流程覆盖

📋 DD尽调 → 📐 方案设计 → 📝 交易文件 → ✅ 交割整合 + 🌏 跨境专项

## 为什么做这个？

做并购案子，尽调清单、SPA 条款、交割核查表……很多都是重复劳动。让 AI 处理这些模板工作，省下的时间用来做真正的专业判断。

**开源、免费、MIT 许可。** 欢迎 Star / Fork / 提 PR 一起完善。

## 怎么装？

```bash
git clone https://github.com/12754271-maker/merger-acquisition-skill.git
cp -r merger-acquisition-skill/skills/ma ~/.qclaw/skills/ma/
```

装完就能用，不需要配置 API Key，不需要付费。
