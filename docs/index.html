📦 Skill 一览

🧠

并购法律工作总控

语义路由 / 事项隔离 / 法规校验 / 质量门调度。作为总入口自动分派到对应的子 Skill。

🔍

并购尽职调查

尽调方案制定、清单生成、发现问题分析、DD 报告撰写。含行业可扩展核查模板。

📐

并购交易方案设计

股权 vs 资产收购对比、税务筹划框架、支付安排设计、交割条件规划。

📝

并购交易文件

SPA / SHA / APA 起草、核心条款谈判要点、附件体系。含买方/卖方双视角条款库。

✅

并购交割与整合

交割条件核查表、交割流程管理、交割备忘录、整合风险评估。

🌏

跨境并购专项

ODI 备案、外资审查、跨境支付、适用法律冲突分析。

💬 这样用

"帮我写一份制造业并购尽调清单"
→ 自动路由到 并购尽职调查
"我是买方，起草 SPA 核心条款"
→ 自动路由到 并购交易文件
"收购一家新加坡公司要什么流程"
→ 自动路由到 跨境并购专项
"股权收购和资产收购怎么选"
→ 自动路由到 并购交易方案设计
"帮我准备交割条件核查表"
→ 自动路由到 并购交割与整合
"做个 XX 公司的并购尽调方案"
→ 总控 判断意图 → 路由到 尽职调查

⚡ 安装

# 1. 克隆仓库
git clone https://github.com/12754271-maker/merger-acquisition-skill.git

# 2. 复制到 QClaw skills 目录
cp -r merger-acquisition-skill/skills/ma ~/.qclaw/skills/ma/

# 3. 编辑你的身份信息
# 编辑 ~/.qclaw/skills/ma/并购法律工作总控/references/practice-profile.md

# 4. 重启 QClaw，开始使用 🎉
