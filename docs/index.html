📦六大模块

🧠

并购法律工作总控

统一入口，语义路由、事项隔离、法规校验、质量门调度。自动识别意图并分派到对应子模块。

🔍

并购尽职调查

尽调方案制定、清单生成、问题分析、DD 报告撰写。含可扩展的行业核查模板体系。

📐

并购交易方案设计

股权 vs 资产收购路径对比、税务筹划框架、支付安排、交割条件规划。

📝

并购交易文件

SPA / SHA / APA 起草，核心条款谈判要点分析。含买方与卖方双视角条款库。

✅

并购交割与整合

交割条件核查表、流程管理、交割备忘录、整合风险评估。

🌏

跨境并购专项

ODI 备案、外资安全审查、跨境支付路径、适用法律冲突分析。

💬使用方式

帮我写一份制造业并购尽调清单
自动路由 → 并购尽职调查
我是买方，起草 SPA 核心条款
自动路由 → 并购交易文件

自动路由 → 跨境并购专项
股权收购和资产收购怎么选
自动路由 → 并购交易方案设计
帮我准备交割条件核查表
